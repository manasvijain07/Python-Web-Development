Welcome!

This is a sample Django user login project. Features include:

  • Custom password validation
  • A password strength “meter” that displays when validating passwords
  • Ability for user to change existing password
  • User profile page with various editable fields that contain custom validation as needed
  • JavaScript date dropdown for the Date of Birth validation feature
  • Basic HTML stripping for text formatting on the "bio" text area
  • Online image editor for the user avatar. Includes basic functionality: rotate, crop and flip
