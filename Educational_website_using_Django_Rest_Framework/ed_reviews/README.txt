This project is all about creating a website for viewing education contents.

As a user I can perform following operations:-

I can Signup ,Login and Logout for this education website.

I can reset password through emailif i forgot my pasword.

I am allowed to access content as per my role(4 roles available:-banned,Member,Moderator,Admin)

I will be redirected to customized error page if i will give bad request.

I can Register for courses.

I can create,update,retrieve and delete courses as per permissions alloted to me.

I will not be allowed to performed update and delete courses for a certain number of times per minute if i am authenticated user.

I will not be allowed to performed update and delete courses for a certain number of times per minute if i am unauthenticated user.
